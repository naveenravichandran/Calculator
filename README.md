# React-Calculator

##### A Simple React Calculator With Hooks and Grid

## Built With

React 16.13.1
React Hooks
Grid

### How To Use:

Step 1:
Navigate into the project folder.

```sh
cd react-calc
```

Step 2:

```sh
git clone https://github.com/naveenravichandran/Calculator.git
```

Step 3:

```sh
npm install
```

Step 4:

```sh
npm start
```
